# inbound-lead-nlp

NLP pipeline for real estate lead response automation.
Walsh College | Masters in AI/ML | Capstone QM640

## Stages
- Stage 1: Intent Classification (DistilBERT vs TF-IDF baseline)
- Stage 2: Urgency Scoring
- Stage 3: RAG-based Response Generation

## Dataset
- Primary training: Zillow real estate conversations (21,347 rows, 108 intent categories)
- Generalisation test: Reddit buyer enquiries (4,038 rows)
